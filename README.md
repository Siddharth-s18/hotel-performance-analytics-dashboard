# Hotel Performance Analytics Dashboard

## Overview
This project analyzes hotel booking data using Power BI to uncover insights related to revenue, bookings, ADR (Average Daily Rate), occupancy, and cancellations.

## Tools Used
- Power BI
- DAX
- Excel / CSV

## KPIs
- Total Revenue: $26M
- Total Bookings: 119K
- Average Daily Rate (ADR): $101.8
- Total Cancellations: 44K
- Cancellation Rate: 37.04%

## Key Insights
- City Hotels generated higher revenue than Resort Hotels.
- Online Travel Agencies contributed the highest revenue.
- Booking volume showed clear seasonal trends.
- Cancellation rate exceeded 37%, highlighting revenue leakage.
- Weekday revenue outperformed weekend revenue.

## Business Impact
The dashboard helps hotel stakeholders monitor performance, identify revenue opportunities, and make data-driven decisions.
